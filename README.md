# 工地材料回收积分系统

## 1. 项目定位

本项目面向施工单位现场管理场景，目标是把“多余材料回收入库”这件事数字化，并通过积分激励机制推动工人、班组主动参与材料节约。

系统第一阶段采用：

- 手机端 `H5`：用于现场快速登记、拍照上传、积分查询、兑换申请
- PC 管理后台：用于基础数据管理、积分规则维护、回收记录审核、兑换审核、报表统计
- 服务端 API：统一处理登录、业务规则、文件上传、积分计算与权限控制

## 2. 当前已确定的技术路线

- 手机端：`Vue 3` + `Vant`
- 管理后台：`Vue 3` + `Element Plus`
- 服务端：`Spring Boot`
- 数据库：`MySQL 8.x`
- 缓存：`Redis`
- 文件存储：本地开发阶段先用服务器磁盘，后续可切换到 `OSS / COS / MinIO`
- 部署：本地开发优先，后续部署到阿里云或腾讯云
