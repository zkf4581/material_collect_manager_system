## 3. 手工测试用例

### 用例 1：新增材料

步骤：

1. 登录后台
2. 进入“材料管理”
3. 输入材料名称、单位、状态
4. 点击保存

预期结果：

- 提示创建成功
- 列表中出现新材料

### 用例 2：编辑材料

步骤：

1. 在材料列表点击“编辑”
2. 修改材料名称或状态
3. 点击保存

预期结果：

- 提示更新成功
- 列表显示更新后的材料信息

### 用例 3：新增积分规则

步骤：

1. 进入“积分规则”
2. 选择材料、单位、完好度
3. 填写基础分值和完好度系数
4. 点击保存

预期结果：

- 提示创建成功
- 规则列表出现新规则

### 用例 4：编辑积分规则

步骤：

1. 在规则列表点击“编辑”
2. 修改分值或系数
3. 点击保存

预期结果：

- 提示更新成功
- 规则列表显示更新后的值
