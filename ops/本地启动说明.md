# 本地启动说明

## 1. 前置条件

- 已安装 `Node.js`
- 已安装 `pnpm`
- 已安装 `Docker`
- 已安装 `Java 17+`

## 2. 启动顺序

### 2.1 启动数据库与缓存

在项目根目录执行：

```bash
docker compose -f ops/docker/docker-compose.local.yml up -d
```

### 2.2 启动服务端

```bash
cd server
./mvnw spring-boot:run
```

### 2.3 启动 H5

```bash
cd client-h5
pnpm install
pnpm dev
```

### 2.4 启动管理后台

```bash
cd admin-web
pnpm install
pnpm dev
```

## 3. 默认地址

- H5：`http://localhost:5173`
- 后台：`http://localhost:5174` 或 Vite 分配端口
- 服务端：`http://localhost:8080`
- 健康检查：`http://localhost:8080/api/health`
